# Security Policy

## Supported versions

This repository is in active research development; security fixes are applied on the default branch.

## Reporting a vulnerability

Please report vulnerabilities privately by emailing the maintainers listed in the project profile.
Do not open public issues for exploitable findings until coordinated disclosure is approved.

Include:
- affected component and commit hash
- impact and exploitation path
- proof-of-concept steps
- suggested mitigation

## Disclosure policy

- Acknowledgement target: 72 hours
- Triage target: 7 days
- Patch/mitigation target: depends on severity and reproducibility
